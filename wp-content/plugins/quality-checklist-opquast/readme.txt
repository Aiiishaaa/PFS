Qualité Web : les bonnes pratiques web dans une checklist de 226 critères de référence

== Description ==

**Accessibilité, référencement, performance, navigation, mobile, vidéo, sécurité… les bonnes pratiques du Web réunies dans la checklist de référence des professionnels du Web !**

Cette extension WordPress vous permet d’obtenir les 226 critères de la checklist Qualité web de Opquast dans votre back-office WordPress.

Vous pouvez cocher les critères conformes, non-conformes ou non-applicables de votre site internet et ainsi obtenir une vue complète du niveau de conformité de votre site en regard des critères de qualité web Opquast.

Cette extension a été réalisée par [Jb Audras](https://jeanbaptisteaudras.com/), directeur technique de l’[agence WordPress Whodunit](https://whodunit.fr), avec la participation de Élie Sloim, président et fondateur d’[Opquast](https://www.opquast.com/).

Les bonnes pratiques Opquast sont proposées sous licence [Creative Commons BY-SA](https://creativecommons.org/licenses/by-sa/2.0/fr/). Les moyens de contrôle et de mise en œuvre sont mis à disposition sous licence [Creative Commons BY-NC-SA](https://creativecommons.org/licenses/by-nc-sa/2.0/fr/).

Cette extension est pour l’instant uniquement disponible en français, les moyens de contrôles et de mise en œuvre des bonnes pratiques Opquast n’étant pas encore 100% traduits en anglais.

== Installation ==

1. Installer l’extension sur votre site WordPress, puis activez-là
2. Allez dans Outils > Checklist Opquast pour utiliser votre checklist
